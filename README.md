# Portfolio Website

A modern portfolio website built with React and Tailwind CSS. This portfolio showcases my skills, experience, and projects in a clean and responsive design.

## Features
- Responsive design
- Modern UI with Tailwind CSS
- Interactive components
- Project showcase
- Contact information

## Technologies Used
- React
- Tailwind CSS
- Vite
- React Icons

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
